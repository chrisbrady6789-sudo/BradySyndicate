⚙️ One-time Setup Required

1
Create Firebase project
Go to console.firebase.google.com → New Project → "brady-syndicate" → Add Web App → copy the config object
2
Enable Realtime Database
In Firebase console → Build → Realtime Database → Create Database → Start in test mode
3
Paste your config
Open this HTML file in a text editor, find the section labelled PASTE FIREBASE CONFIG HERE near the bottom and replace the placeholder object
4
EmailJS (optional — for email notifications)
Go to emailjs.com → free account → create a service + template → paste your Service ID, Template ID, and Public Key into the EMAILJS CONFIG section
5
Discord webhook (optional)
In your Discord server → channel settings → Integrations → Webhooks → New Webhook → copy URL → paste into DISCORD_WEBHOOK below
6
Enable Firebase Storage (for bet screenshots)
Firebase console → Build → Storage → Get started → Start in test mode. Then update Storage Rules to:
rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /bet-screenshots/{allPaths=**} {
      allow read, write: if true;
    }
  }
}
